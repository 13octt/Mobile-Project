
# IOT of UIT 

This project is about developing an application that can observe the weather of the UIT


## Authors

- [@Tzuy284](https://github.com/Tzuy284)
- [@13octt](https://github.com/13octt)


## Features

- Monitor the weather 
- Weather index statistics
- Create study schedule 
- Map
